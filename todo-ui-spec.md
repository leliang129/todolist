# Todo List 界面设计需求整理

> 说明：以下为基于当前需求与 AI 生成示意图的整理版本（示意图参考对话中的截图）。

## 1. 整体布局
- 布局结构：左侧固定侧边栏（240px）+ 右侧主内容区（占满剩余宽度）
- 交互原则：无页面跳转，所有功能在同一界面完成

## 2. 左侧侧边栏（固定，可折叠）
- 背景：浅灰 `#F5F7FA`
- hover 状态：浅蓝背景
- 从上至下的模块结构：

### 2.1 顶部 Logo / 标题
- 文案：**Todo List**
- 样式：粗体，主色 `#4A90E2`，字号 `20px`

### 2.2 分类模块（核心）
#### 快捷分类
- 全部待办
- 待完成
- 已完成
- 今日待办
- 快到期
- 右侧数字角标：显示对应数量

#### 自定义分类
- 默认：我的工作、生活琐事、学习计划
- 功能：可新增 / 编辑 / 删除
- 视觉：左侧带分类小色块

### 2.3 统计模块
- 极简数据展示：
  - 待办总数
  - 今日完成数
  - 本周完成率
- 表现形式：文字 + 简易百分比条

### 2.4 底部入口
- 设置（齿轮图标）
- 意见反馈
- 关于

## 3. 右侧主内容区

### 3.1 顶部操作栏（固定）
- 背景：白色
- 效果：轻微阴影
- 从左至右布局：

#### 搜索框
- 圆角 `8px`
- 占位文案：**搜索待办（标题 / 标签）**
- 左侧图标：放大镜

#### 功能按钮
- **新增待办**（主按钮）
  - 主色背景
  - 白色文字
  - 圆角 `8px`
  - “+” 图标 + 文字
- **筛选**（轮廓按钮）
  - 主色边框
  - 触发下拉 / 抽屉
- **排序**（轮廓按钮）
  - 主色边框
  - 下拉菜单：按创建时间 / 截止日期 / 优先级

#### 右侧操作
- **清空已完成**：文字按钮，浅灰色
- 点击后：弹窗确认

### 3.2 待办列表区
- 背景：白色
- 列表项底色：`#F5F7FA` 交替
- 无滚动时：居中提示
  - 文案：**暂无待办，点击新增开始吧～**

### 3.3 单条待办项布局
- 高度：`64px`
- 内边距：左右 `16px`
- 垂直居中
- hover：轻微阴影
- 从左至右结构：

#### 完成按钮
- 形态：圆形复选框
- 未完成：白色边框
- 已完成：绿色底 + 对勾
- 已完成文字：浅灰 + 划线

#### 核心内容（占比约 70%）
- 标题：
  - 字号 `16px`
  - 加粗
  - 颜色 `#333333`
  - 完成态：`#8C8C8C` + 划线
- 副文本（单行，`12px`，`#666666`）：
  - 优先级标识（色块 + 文字，如“高”）
  - 截止日期（快到期标橙红，已过期标红，无则不显示）
  - 分类标签（小色块 + 文字）
  - 分隔形式：` / `

#### 操作按钮（hover 显示）
- 默认隐藏，避免界面杂乱
- **编辑**（铅笔图标）
- **删除**（垃圾桶图标）
- 图标颜色：`#666666`
- hover：编辑变主色 / 删除变红色

### 3.4 空状态 / 完成态提示
- 无待办：
  - 居中显示 **“+ 新增待办”** 轻量图标 + 文字
  - 主色
- 全部完成：
  - 居中显示 **“今日任务已完成，休息一下吧～”**
  - 庆祝小图标
  - 浅绿色

## 4. 技术栈选型
- 前端：React 18 + Vite + Ant Design
- 后端：Python + FastAPI + MySQL
